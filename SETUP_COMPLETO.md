## 🆘 PROBLEMAS COMUNS

| Problema | Solução |
|----------|---------|
| JSON não carrega | Valide em jsonlint.com |
| Imagens não aparecem | Verifique o caminho: `assets/images/` |
| Cores não mudam | Use formato hexadecimal: `#RRGGBB` |
| Site lento | Comprima imagens com TinyPNG |
| Links não funcionam | Verifique `https://` no início |
| Modal não abre | Abra console (F12) para erros |
